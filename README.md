# \# POS System

# 

# \## Requirements

# \- Node.js v18+

# \- PostgreSQL Database (Supabase recommended)

# 

# \## Backend Setup

# 1\. Open terminal and go to backend folder: `cd backend`

# 2\. Install dependencies: `npm install`

# 3\. Copy `.env.example` and rename it to `.env`

# 4\. Fill in your values in `.env` file:

# &nbsp;  - PORT: 5000

# &nbsp;  - DATABASE\_URL: your PostgreSQL connection string

# &nbsp;  - JWT\_SECRET: any random secret key

# 5\. Run the backend: `npm run dev`

# 

# \## Frontend Setup

# 1\. Open new terminal and go to frontend folder: `cd frontend`

# 2\. Install dependencies: `npm install`

# 3\. Run the frontend: `npm run dev`

# 

# \## Database Setup

# 1\. Open your Supabase project

# 2\. Go to SQL Editor

# 3\. Open the `database\_dump.sql` file

# 4\. Copy all content and paste it in SQL Editor

# 5\. Click Run

# 

# \## Access

# \- Open browser and go to: `http://localhost:5173`

# \- Login with your Super Admin credentials
